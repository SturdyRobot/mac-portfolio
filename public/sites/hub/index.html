Hi, I’m Noel — “nlj” 🤖

I build deterministic Rust runtimes, WASM execution engines, and AI infrastructure — high-assurance systems from the kernel to the edge. I ship production software solo: I own the architecture, and direct AI to implement against contracts I design and verify.

Everything pinned below is real and runs. The retro desktop you’re looking at this through is one of them.

How I work

Design the contracts myself → let AI draft against them → test, review, and reject what’s wrong → verify every change in a real browser before it lands. The tooling I build reflects the same idea: Kedge runs AI agents under hard budgets with byte-identical replay, so a change’s effect is verifiable, not vibes.
