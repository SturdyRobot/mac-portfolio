How I work

Design the contracts myself → let AI draft against them → test, review, and reject what’s wrong → verify every change in a real browser before it lands. The tooling I build reflects the same idea: SturdyHarness runs AI agents under hard budgets with byte-identical replay, so a change’s effect is verifiable, not vibes.
